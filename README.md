# Eastern Garden Chinese Restaurant Menu

A beautiful, responsive restaurant menu webpage for Eastern Garden Chinese Restaurant.

## Features

- Elegant design with gold and cream color scheme
- Beautiful Chinese food background images
- Fully mobile responsive (works on all devices)
- No JavaScript required - pure HTML & CSS
- Fast loading and SEO-friendly

## Tech Stack

- HTML5
- CSS3 (with CSS Grid, Flexbox, and Media Queries)
- SVG for decorative elements

## How to Use

Simply open `index.html` in any modern web browser.
